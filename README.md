# MD_into_HTML_converter
Converting MD into HTML.

## ABOUT
You can convert Markdown into HTML like fitting the format you customized through this program.

And, you can use all classes in this code.

I really hope you make use of it with Lexical analysis, etc.

## NOTE

If you want more exactly converter, you create it by yourself.
